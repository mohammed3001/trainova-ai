# Trainova AI — وثيقة المنتج الرسمية

**الاسم التجاري:** Trainova AI
**الشعار (Tagline):** *The Global Marketplace for AI Training Talent*
**الإصدار:** 1.0 — وثيقة المنتج الأساسية (Product Requirements Document)
**النوع:** منصة B2B عالمية — Marketplace + Workflow + AI Evaluation

---

## 1. الملخص التنفيذي (Executive Summary)

**Trainova AI** منصة عالمية متخصصة تربط بين الشركات التي تحتاج تدريب أو تحسين أو اختبار نماذج الذكاء الاصطناعي، وبين الخبراء والمدربين القادرين على تنفيذ هذا العمل بمعايير دقيقة.

تختلف المنصة جوهرياً عن منصات العمل الحر وعن أدلة الخبراء التقليدية، لأنها تدمج ثلاثة منتجات في منتج واحد:

1. **Marketplace** للاكتشاف والمطابقة.
2. **Evaluation Engine** لاختبار المتقدمين عمليًا على نموذج الشركة الفعلي أو بيئة مشابهة.
3. **Collaboration Workspace** لإدارة التفاوض والعقد والتنفيذ والدفع.

**الهدف التجاري:** أن تصبح Trainova AI الطبقة المعتمدة (Trust Layer) عالميًا لتوظيف وتقييم خبراء تدريب نماذج الذكاء الاصطناعي، ومصدر دخل دائم عبر: اشتراكات + عمولات + إعلانات متخصصة + خدمات Premium.

---

## 2. المشكلة (Problem)

- توسّع غير مسبوق في اعتماد الذكاء الاصطناعي داخل الشركات، يقابله **نقص حاد في خبراء التدريب والتقييم الموثوقين**.
- منصات العمل الحر العامة (Upwork, Fiverr) لا توفر:
  - اختبار عملي مرتبط بنموذج الشركة.
  - معايير تقييم دقيقة لتخصصات مثل RLHF, Fine-tuning, Prompt Engineering, Safety Alignment.
  - بنية تعاقدية وعمل مؤسسي (NDA, MSA, SOW, Escrow).
- التوظيف المباشر بطيء ومكلف، وشبكات LinkedIn لا تضمن الجودة الفعلية.
- لا توجد اليوم منصة عالمية واحدة جامعة لتخصصات **AI Training**.

**النتيجة:** الشركات تخسر أشهرًا في البحث وتدفع مبالغ ضخمة لوسطاء، والخبراء يعانون صعوبة في الوصول لعقود مؤسسية حقيقية.

---

## 3. الحل (Solution)

**Trainova AI** تعالج هذه المشكلة عبر منصة واحدة متكاملة تقدّم:

1. **نشر ذكي للطلبات**: نموذج مرن يسمح للشركة بتخصيص كل شرط — من المهارات إلى الصناعة إلى نوع النموذج.
2. **اختبار عملي حقيقي**: الشركة تربط API / Sandbox / بيئة تقييم، ويتم اختبار المدرب فعليًا على المهمة.
3. **مطابقة ذكية (AI Matching)**: خوارزمية تقترح أفضل المرشحين بناءً على المهارات، الأداء السابق، ونتائج الاختبار.
4. **سير عمل متكامل**: دردشة احترافية، تفاوض، عقد رقمي، دفع، تقييم نهائي.
5. **ثقة مؤسسية**: توثيق هوية، توثيق خبرة، شارات، مراجعات، escrow.

## 5. المنتج — العناصر الرئيسية (Core Product Modules)

### 5.1 واجهة عامة (Public Website + SEO Engine)
- الصفحة الرئيسية، كيف نعمل، صفحات للشركات/للمدربين.
- دليل المهارات، دليل الخبراء، دليل الطلبات المفتوحة.
- مدوّنة + مركز مساعدة + صفحات SEO ديناميكية تلقائية.
- توليد ميتاداتا وScheme تلقائي + Sitemap تلقائي + FAQ Schema.

### 5.2 بوابة الشركة (Company Portal)
- إنشاء حساب مؤسسي + توثيق.
- نشر طلبات متقدمة بحقول قابلة للتخصيص الكامل.
- ربط API / Sandbox / Endpoint للاختبار الحي.
- بناء اختبارات (MCQ / نصي / عملي مرتبط بالنموذج).
- مراجعة المتقدمين وتصفيتهم.
- AI Matching للترشيحات.
- دردشة احترافية متصلة بالمشروع.
- إدارة العقود والفواتير ومؤشرات الأداء.

### 5.3 بوابة المدرب (Trainer Portal)
- ملف شخصي احترافي: مهارات، خبرات، شهادات، Portfolio.
- بطاقة أعمال قابلة للمشاركة.
- تقديم على الطلبات، تنفيذ الاختبارات.
- إدارة المحادثات والمشاريع والدفعات.
- شارات توثيق واعتماد.

### 5.4 محرك الاختبارات (Evaluation Engine)
- **اختبار نظري**: MCQ + أسئلة نصية + حالات دراسية.
- **اختبار عملي**: كتابة Prompts، تحسين ردود، تصنيف بيانات، بناء workflow.
- **اختبار حي** على API/Sandbox مرتبط من الشركة.
- **تقييم متعدد المراحل**: فرز → عملي → مقابلة → قبول.
- **Scoring**: آلي + يدوي + هجين.
- **معايير تقييم**: دقة، وقت، استهلاك توكنز، جودة المخرجات، الالتزام بالتعليمات.

### 5.5 نظام الدردشة (Collaboration Workspace)
- محادثة 1-1 ومجموعات مرتبطة بالمشروع.
- إرسال ملفات، مشاركة نتائج اختبار، قوالب رسائل.
- تنبيهات، قراءة، أرشفة، بحث داخلي.
- ترجمة فورية اختيارية (AR/EN على الأقل).
- تلخيص AI للمحادثات واستخراج المهام تلقائيًا.
- مكالمات صوت/فيديو (مرحلة ثانية).

### 5.6 نظام المطابقة الذكي (AI Matching Engine)
- يقارن بين المدرب والطلب في: المهارات، الخبرات، التقييمات، نتائج الاختبارات، الميزانية، اللغة، سرعة الإنجاز، نوع النموذج، الصناعة.
- يعرض: أفضل 5/10 مرشحين، نسبة التطابق، نقاط القوة، نقاط المخاطر.
- يستخدم تمثيلات vector + معايير موزونة قابلة للضبط من الأدمن.

### 5.7 نظام العقود والدفع
- NDA / MSA / SOW قوالب جاهزة + تعديل.
- توقيع إلكتروني.
- Escrow + Milestones.
- عمولة منصة قابلة للتخصيص.
- دعم عدة عملات + VAT.

### 5.8 نظام الإعلانات (Ads Platform)
- إعلانات متخصصة مناسبة للجمهور: Featured Trainer/Company، Sponsored Listings، Sidebar، Category Sponsorship، Homepage Sponsor، Newsletter Ads.
- لوحة للمعلنين + تقارير نقر + استهداف حسب الفئة والدولة.

### 5.9 لوحة الأدمن الشاملة (No-Code Mission Control)
تصمم لمالك غير تقني تمامًا، وتشمل:
- Dashboard KPIs شامل.
- إدارة المستخدمين (شركات/مدربين/أدمن فرعي).
- إدارة الطلبات والاختبارات.
- إدارة الإعلانات والاشتراكات.
- Finance Center (عمولات، فواتير، سحوبات، استرداد).
- CMS للصفحات والمقالات والأسئلة الشائعة.
- SEO Center لضبط Meta/Schema/Sitemap.
- Email Center: قوالب، حملات، شرائح، A/B.
- إعدادات الشعار، الألوان، العملات، اللغات، الدول، الرسوم، طرق الدفع.
- نظام صلاحيات متعدد (Super Admin, Finance, Support, Content, Ads).
- تقارير وتحليلات عميقة.

## 6. نموذج الأعمال (Business Model)

### 6.1 الاشتراكات (Subscriptions)
| الفئة | Free | Pro | Enterprise |
|--------|------|-----|------------|
| شركات | نشر محدود | طلبات غير محدودة + AI Matching | SSO + API + Dedicated Manager + Custom Contracts |
| مدربون | ملف أساسي | Verified Pro + Visibility Boost | Premium Placement + Priority Support |

### 6.2 العمولات (Transaction Fees)
- نسبة من كل عقد يُنجز داخل المنصة (5%-15% حسب الباقة).

### 6.3 خدمات Premium
- توثيق متقدم، فحص خبرات، AI Assistant لصياغة الطلبات، Managed Hiring.

### 6.4 الإعلانات
- CPM / CPC / Flat Placements.
- Featured Listings + Sponsored Search + Category Sponsorships.

### 6.5 خدمات Enterprise
- White-label portal، API Access، تكاملات خاصة.

## 7. المزايا التنافسية (Competitive Advantages)

1. **Model-Linked Evaluation** — لا يوجد منافس يربط اختبار المتقدم بنموذج الشركة الفعلي.
2. **Vertical Focus** — تخصص دقيق في AI Training بدل Generalist Freelance.
3. **Trust Layer** — توثيق متعدد الطبقات + Escrow.
4. **AI-Native Platform** — استخدام AI داخلي في SEO، Matching، Profile Optimization، Chat Summaries، Test Generation.
5. **No-Code Admin** — ملاك غير تقنيين يديرون المنصة بالكامل.
6. **Global-First** — i18n AR/EN + RTL/LTR من اليوم الأول، قابل للتوسع لأي لغة.

- معدل تحويل التقديم → قبول.
- CAC / LTV على كل شريحة.
- Net Revenue Retention للشركات Pro/Enterprise.
